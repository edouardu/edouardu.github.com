Ongoing Prototyping

Practice Sandbox
Day–to–Day
2013 — Ongoing

A crazy thing about sotware is that tools for making it are constantly being improved, made obsolete, forked, and iterated upon. Frameworks and Libraries sprout left and right, and you never really know what's best for what at any given time. This repository is for practicing with these things, and it's always a mess. Proceed with caution!

How 2 Github Pages
Simple Tutorial Site
2015

While I was at IBM, I made it a personal mission to teach designers that Git(hub) was the shit — both the technology and the popular platform. An often repeated question many had was, “How do I make and host web-based prototypes?” This website I spun up in less than 2 minutes was the answer I would give, and later attach to a Slack shortcut. Later on, I threw in confetti I stole from David Rudnick's site http://trance.army/. It's good stuff.

Reference SS/FW
Archival Project
2014 — Ongoing

Sometime in 2014 I got really sick of reblogging already-circulating media back into the void and decided I was going to be a lot more intentional and thoughtful with the visual material I collected across various places on the web. I decided to use Github, Github Releases, and Git LFS to maintain a collection of particularly affecting material I come across throughout the year, as well as snippets of my own work process for anyone who cares. I’m scheduling these drops twice a year, during the typical Spring/Summer and Winter/Fall time periods new clothing is showcased — just cuz.

Résumé
Text Adventure Game
2014

I had an incredibly wacky idea a few years ago to make my website/résumé into an interactive text adventure. The idea was that you'd chat to a fictional version of me and learn about my work/ethos/whatever in a conversational tone. You could enact violence upon me to end the experience. This whole thing was built in Twine and heavily inspired my one of my biggest inspirations/mentors, Anna Anthropy. Source is here.

Snapshots
Raspberry Pi Installation
2014

A project inspired by a currently-defunct undertaking by Jefferey Scudder. The original project comprised of 3-ish second long webcam videos taken the same time every day. What resulted was a string of youtube videos depicting life as it passed by in front of a computer. I thought it would be cool to mimic this project with a remotely-controlled Raspberry Pi and camera set to go off at 11am and 11pm every day with a simple cron job script. Right now my project is also defunct, but I plan on reinstating it soon.
